O Projeto

JavaFX é uma recente biblioteca para criação de interfaces gráficas, para programas desenvolvidos usando a linguagem Java.

Com a JavaFX fica mais fácil a criação e manipulação de elementos 2D e 3D, audio, video, animação, webview, entre outros recursos gráficos. Com esta biblioteca podemos desevolver programas tanto para desktop quanto para mobile.

O Objetivo desta tour, é passar pelas universidades e ensinar aos alunos o paradigma Orientação a Objetos e como usar este paradigma para criação de programas com Interfaces Gráficas em JavaFX.

Gratuito

O mini curso é gratuito e tem carga horária de 4 horas.

Certificado

O curso terá certificado de 4 horas de atividades complementares.

Ementa

  • Introdução ao JavaFX
    • Histórico
    • A Metáfora da Cena Gráfica
    • Paradigma Orientação a Objetos
  • Construindo Programas JavaFX
    • Construção da Cena Gráfica
    • Trabalhando com objetos gráficos
    • Importando e utilizando arquivos CSS
    • Manipulação de Mídia: áudio, vídeo e imagens
    • Animações 2D e 3D
